How long do I endure?

The interactive chart below displays the amount of time I have spent running, cycling, and pursuing other endurance sports such as swimming and hiking.

Click and drag on the canvas area behind the plot to focus on a smaller range of time. Double click the canvas to reset. Interactive features work best on desktop.

Data is automatically fetched from Strava using the Strava API and GitHub Actions. Visit my GitHub repository to see how I accomplish this.
